# Pomodoro Timer App

A simple web-based Pomodoro Timer application built with HTML, CSS, and Vanilla JavaScript, bundled using Parcel.

## Features

- **25-minute Pomodoro Timer:** The core feature for standard focus sessions.
- **Short Break (5m):** A quick break interval option.
- **Long Break (15m):** A longer break interval option for extended rest.
- **Start/Pause/Reset Controls:** Easy to use timer controls.
- Clean and modern user interface.

## Project Structure

- `src/index.html`: Main HTML entry point.
- `src/style.css`: Styles for the UI.
- `src/app.js`: timer logic and DOM interactions.
- `package.json`: Project dependencies and scripts.

## Usage

Once the app is running (see DEVELOPMENT.md for how to run), you will be presented with a timer. You can select between "Pomodoro", "Short Break", or "Long Break" to set the time, and use the "Start", "Pause", and "Reset" buttons to control the timer flow.
